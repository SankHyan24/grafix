## Explicit Representations
1. Point Cloud (Explicit)
    - Easiest representation: list of points(x,y,z)
    - 只要点足够密，表示任何面
    - 经常被转化成多边形面
2. Polygon Mesh (Explicit)
    - Store vertices & polygons(一般是三角形/四边形)
    - 更复杂的数据结构，更普遍的表达形式
3. The Wavefront Object File (.obj) Format
